## IBM Swift theme

A theme for Atom.

Inspired by IBM's [Swift Sandbox Theme](https://developer.ibm.com/swift/2015/12/03/introducing-the-ibm-swift-sandbox/). The code is based on the [Firewatch Syntax Theme](https://atom.io/themes/firewatch-syntax).
